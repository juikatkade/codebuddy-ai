# 🚀 CodeBuddy AI

> **Your AI-Powered Coding Companion for LeetCode**

CodeBuddy AI is a Chrome Extension built to enhance your **LeetCode** experience. Whether you're preparing for coding interviews or improving your problem-solving skills, CodeBuddy AI provides intelligent AI assistance, productivity tools, and focus features to help you learn more effectively.

## ✨ Features

### 🤖 AI-Powered Hints

* Get contextual AI hints without revealing the complete solution.
* Learn the approach step-by-step instead of copying answers.
* Improve your problem-solving skills with guided assistance.

### 🎯 Focus Mode

* Stay focused while solving problems.
* Track coding sessions.
* Minimize distractions during practice.

### 📊 Session Analytics

Track your progress with:

* Coding session duration
* Number of problems solved
* Productivity insights

### 🔔 Smart Notifications

Receive browser notifications for:

* Focus session reminders
* Productivity updates
* Upcoming features like LeetCode contest reminders

### 🔐 Secure API Key Management

* Bring your own Google Gemini API Key.
* Your API key is securely stored using Chrome Extension Storage.
* No API keys are hardcoded.

## ⚙️ How to Use

1. Install the extension from the Chrome Web Store.
2. Open any LeetCode problem.
3. Open the CodeBuddy AI extension.
4. Enter your Google Gemini API Key.
5. Start solving problems with AI-powered guidance.

---

## 🔒 Privacy

CodeBuddy AI only accesses LeetCode pages to provide its features.

User preferences and API keys are stored securely using Chrome Extension Storage.

The extension does not sell or share personal data for advertising purposes.
